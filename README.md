# LEGO Robotic Team 12 - 2025/2026

This repository contains the project materials for **Team 12** in the **Robotics M** course at the **University of Glasgow**.

We are students from the **Robotics & AI** programme, working on a LEGO robotics project as part of the Robotics M module.

## Project Overview

This project focuses on the design, development, testing, and evaluation of a LEGO robot for the Robotics M coursework. The repository includes project targets, materials, reports, and other relevant documentation produced during the project.
